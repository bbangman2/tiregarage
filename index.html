🗑️ 관리자 — 메시지 삭제
이호준 전용 기능입니다. 삭제된 메시지는 복구 불가합니다.
📅 날짜별 삭제
~
선택한 날짜(들)의 데이터를 삭제합니다. 입고방·직배송방·채팅 모두 해당 방의 데이터를 지웁니다.
💥 방 전체 초기화
해당 방 전체 메시지를 완전히 초기화합니다.
💾 백업 (CSV 다운로드)
방을 선택해 데이터를 CSV로 저장합니다. 엑셀에서 열 수 있어요.
~
📤 복원 (CSV 업로드)
백업한 CSV를 다시 불러옵니다. 실수로 지운 데이터를 되살릴 때 쓰세요.
건너뛰기=이미 있는 건 그대로 둠 / 덮어쓰기=CSV의 날짜 데이터를 비우고 새로 넣음
받아오기 분류 단어
출처에 이 단어가 포함되면 종합본에서 '받아오기'(직접 가져올 것)로 분류돼 맨 위에 모입니다.
거래처발주 단어 (정확히 일치)
출처가 이 단어와 정확히 같으면 '거래처발주'로 분류됩니다. 예: 매일오전, 넥센, 한타, 극동
물류차입고 단어 (정확히 일치)
출처가 이 단어와 정확히 같으면 '물류차입고'로 분류됩니다. 예: 넥센물류, 한국물류
반품 분류 단어 (포함)
출처에 이 단어가 포함되면 '반품'으로 분류돼 메모탭에 따로 표시됩니다. 예: 극동반품, 한국반납
⚙️ AI 정리 설정
