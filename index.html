📅 날짜별 삭제
~
선택한 날짜(들)의 데이터를 삭제합니다. 입고방·직배송방·채팅 모두 해당 방의 데이터를 지웁니다.
💥 방 전체 초기화

💾 백업 (CSV 다운로드)
방을 선택해 데이터를 CSV로 저장합니다. 엑셀에서 열 수 있어요.
~
📤 복원 (CSV 업로드)
백업한 CSV를 다시 불러옵니다. 실수로 지운 데이터를 되살릴 때 쓰세요.
건너뛰기=이미 있는 건 그대로 둠 / 덮어쓰기=CSV의 날짜 데이터를 비우고 새로 넣음
📋 출처 관리
출처를 등록하고 각각 어디로 분류될지 정합니다. 여기 등록된 순서대로 입고방 출처 드롭다운에 나옵니다.
🏷️ 분류(카테고리) 관리
재고표의 분류 칸에 쓸 수 있는 단어들. 추가/삭제·순서변경(드래그) 가능. 새 분류를 셀에 입력하면 자동으로 여기에 등록됩니다.
📄 업로드 양식 다운로드
엑셀 양식을 받아서 채운 뒤, 재고방의 ➕ 행 추가 → 붙여넣기 또는 📥 업로드 메뉴로 올리세요.
반품 분류 단어 (포함)
출처에 이 단어가 포함되면 '반품'으로 분류돼 메모탭에 따로 표시됩니다. 예: 극동반품, 한국반납
