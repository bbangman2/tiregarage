📥 종합본 엑셀 다운로드
날짜 범위를 골라 입고표를 엑셀(.xlsx)로 받습니다. 값으로 저장돼 기록이 그대로 보존됩니다. 수정·삭제 없이 다운로드만 합니다.
🗑️ 관리자 — 메시지 삭제
이호준 전용 기능입니다. 삭제된 메시지는 복구 불가합니다.
📅 날짜별 삭제
~
선택한 날짜(들)의 데이터를 삭제합니다. 입고방·직배송방·채팅 모두 해당 방의 데이터를 지웁니다.
💥 방 전체 초기화
해당 방 전체 메시지를 완전히 초기화합니다.
💾 백업 (CSV 다운로드)
방을 선택해 데이터를 CSV로 저장합니다. 엑셀에서 열 수 있어요.
~
📤 복원 (CSV 업로드)
백업한 CSV를 다시 불러옵니다. 실수로 지운 데이터를 되살릴 때 쓰세요.
건너뛰기=이미 있는 건 그대로 둠 / 덮어쓰기=CSV의 날짜 데이터를 비우고 새로 넣음
📋 출처 관리
출처를 등록하고 각각 어디로 분류될지 정합니다. 여기 등록된 순서대로 입고방 출처 드롭다운에 나옵니다.
🧩 모델명 사전
종합방→입고방으로 넘길 때 상품명에서 모델 코드를 뽑아 짧게 표기합니다(예: 솔루스 TA21 → TA21). 대소문자는 전부 대문자로 통일됩니다. 사전에 없으면 원문(대문자)으로 표시됩니다. 한 제품을 여러 이름으로 부르면(예: 슈프림=NPS=NF SUPREME) 별칭에 쉼표로 넣으세요.
📤 전체 교체(초기화 후 업로드)
한 줄에 하나씩. 대표코드만 쓰거나, 별칭이 있으면 대표코드 = 별칭1, 별칭2 형식. 예:
TA21
슈프림 = NPS, NF SUPREME, SUPREME
💰 거래처 종합방 권한
이 명단에 있는 사람에게만 거래처 종합방이 보입니다(대표는 항상). 방 접근 권한자는 잠금·편집·삭제가 모두 자유롭습니다.
🏷️ 분류(카테고리) 관리
재고표의 분류 칸에 쓸 수 있는 단어들. 추가/삭제·순서변경(드래그) 가능. 새 분류를 셀에 입력하면 자동으로 여기에 등록됩니다.
📄 업로드 양식 다운로드
엑셀 양식을 받아서 채운 뒤, 재고방의 ➕ 행 추가 → 붙여넣기 또는 📥 업로드 메뉴로 올리세요.
반품 분류 단어 (포함)
출처에 이 단어가 포함되면 '반품'으로 분류돼 메모탭에 따로 표시됩니다. 예: 극동반품, 한국반납
⚙️ AI 정리 설정
Claude API 키 등록 시 AI가 채팅을 직접 이해해서 정리합니다.
미등록 시 패턴 자동 정리(무료)로 동작합니다.

👥 재고방 수정권한

🔑 로그인 비밀번호 관리
